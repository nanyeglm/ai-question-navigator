简短介绍python

Python 是一种高级、解释型、通用编程语言，以简洁易读的语法著称。由Guido van Rossum于1989年创建，支持多种编程范式，包括面向对象、函数式和过程式编程。广泛应用于Web开发、数据分析、人工智能、自动化脚本等领域。Python拥有丰富的标准库和第三方库（如NumPy、Pandas、TensorFlow），社区活跃，适合初学者和专业开发者。当前最新版本为Python 3.x。

简短介绍NumPy

NumPy 是 Python 的一个开源科学计算库，专注于高效的多维数组（ndarray）操作和数学计算。它提供快速的数组运算、线性代数、统计分析和随机数生成等功能，广泛用于数据科学、机器学习和工程计算。NumPy 的核心是基于 C 实现的，性能优异，常作为 Pandas、SciPy 等库的基础。
